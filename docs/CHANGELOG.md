## [1.0.0] - 2026-06-22

### 新增
- 从 2D Canvas 迁移到 Three.js 3D 引擎
- 实现第一人称视角控制（WASD + 鼠标）
- 添加 PointerLockControls 支持
- 实现多层星空背景系统
- 添加程序化行星系统（含大气层和行星环）
- 实现体积感星云效果
- 添加后处理效果（辉光、暗角、色差）
- 创建 HUD 界面（准星、FPS、速度、位置）
- 添加加载画面
- 完善操作提示

### 技术特性
- 使用 InstancedMesh 优化大量星星渲染
- 自定义 Shader 实现大气层和星云效果
- EffectComposer 后处理管线
- 模块化架构设计
- 完整的配置系统

### 控制方式
- W/A/S/D: 前后左右移动
- 鼠标: 控制视角
- 空格: 上升
- Shift: 下降
- Ctrl: 冲刺
- 点击屏幕: 锁定/解锁鼠标

### 配置参数
- 移动速度: 50（可调）
- 冲刺倍数: 2.5x
- 星星数量: 8000
- 行星数量: 8
- 辉光强度: 1.5
